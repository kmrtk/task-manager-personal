# 要件定義書　タスク管理アプリ（Trello風）

---

## ① 使用目的
講義の課題として作成するTrello風タスク管理アプリ。自身のタスクを管理できるアプリケーションとして活用することを目的とする。

---

## ② 対象ユーザー・ユースケース

**対象ユーザー：** 自分（将来的にチーム向けに拡張する可能性あり）

**ユースケース：**

| ユースケース | 機能要件 |
|------------|---------|
| フォルダ（仕事・プライベートなど）を作成・管理する | F01, F02, F03 |
| フォルダ内にグループ（未着手・進行中など）を追加・並べ替える | F04, F05, F06, F07 |
| タスクを作成する | F08, F09, F31 |
| タスクを別のグループに移動する | F11, F12 |
| タスクに締め切りを設定して期限切れを確認する | F17, F18, F19, F20 |
| タスクにタグを付けて分類する | F13, F14, F15, F16 |
| タスクの優先度を設定する | F21 |
| タスクにメモやサブタスクを追加する | F22, F23, F24, F25, F26, F27 |
| タスクを削除・元に戻す | F10, F28, F29, F30 |

## ④ 機能要件

### ボード
| ID | 機能名 | 内容 |
|----|--------|------|
| F01 | ボード作成 | 新規ボードを作成できる |
| F02 | ボード編集 | ボード名を変更できる |
| F03 | ボード削除 | ボードを削除できる |

### カラム
| ID | 機能名 | 内容 |
|----|--------|------|
| F04 | カラム作成 | ボード内にカラムを追加できる |
| F05 | カラム編集 | カラム名を変更できる |
| F06 | カラム削除 | カラムを削除できる |
| F07 | カラム並び替え | カラムの順番をドラッグ＆ドロップで変更できる |

### カード
| ID | 機能名 | 内容 |
|----|--------|------|
| F08 | カード作成 | カラム内にカードを追加できる |
| F09 | カード編集 | カードのタイトル・内容を編集できる |
| F10 | カード削除 | カードをゴミ箱に移動できる |
| F11 | カード移動 | カードをカラム間でドラッグ＆ドロップで移動できる |
| F12 | カード並び替え | カラム内のカードの順番を変更できる |

### ラベル
| ID | 機能名 | 内容 |
|----|--------|------|
| F13 | ラベル作成 | ラベルを作成できる |
| F14 | ラベル編集 | ラベル名・色を変更できる |
| F15 | ラベル削除 | ラベルを削除できる |
| F16 | ラベル付与 | カードにラベルを付ける・外すことができる |

### 期限・アラート
| ID | 機能名 | 内容 |
|----|--------|------|
| F17 | 期限設定 | カードに期限日をカレンダーUIで設定・編集・削除できる |
| F18 | 期限アラート（黄） | 期限1週間以内のカードを黄色で表示する |
| F19 | 期限アラート（赤） | 期限切れのカードを赤で表示する |
| F20 | 完了表示 | 完了したカードを灰色で表示する（手動操作） |

### 優先度
| ID | 機能名 | 内容 |
|----|--------|------|
| F21 | 優先度設定 | カードに優先度（色）を手動で設定・変更・削除できる |

### コメント
| ID | 機能名 | 内容 |
|----|--------|------|
| F22 | コメント追加 | カードにコメントを追加できる |
| F23 | コメント編集 | コメントを編集できる |
| F24 | コメント削除 | コメントを削除できる |

### チェックリスト
| ID | 機能名 | 内容 |
|----|--------|------|
| F25 | チェックリスト作成 | カード内にチェックリストを追加できる |
| F26 | 項目管理 | チェックリストの項目を追加・編集・削除できる |
| F27 | チェック操作 | 項目にチェックを入れる・外すことができる |

### ゴミ箱
| ID | 機能名 | 内容 |
|----|--------|------|
| F28 | ゴミ箱確認 | 削除したカードをゴミ箱で一覧表示できる |
| F29 | カード復元 | ゴミ箱からカードを元のカラムに戻せる |
| F30 | 完全削除 | ゴミ箱のカードを完全に削除できる |

### 日付
| ID | 機能名 | 内容 |
|----|--------|------|
| F31 | 作成日自動記録 | カード作成時に作成日を自動で記録・表示する |
